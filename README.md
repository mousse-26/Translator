# 🌐 Text Translator using Google Translate API

A simple Python-based text translator that allows users to translate text between multiple languages using the `googletrans` library.

---

## 🚀 Features
- 🌍 Translate text between multiple languages  
- 📝 Supports language selection using language codes  
- 📚 Displays all supported languages  
- ⚡ Simple and easy-to-use command-line interface  

---

## 🛠️ Tech Stack
- Python  
- googletrans  

---

💡 How It Works
User inputs the text to be translated
The program displays supported languages
User selects source and destination language codes
Text is translated using the Google Translate API
Output is displayed in the terminal
